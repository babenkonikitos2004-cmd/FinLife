# Руководство по оптимизации изображений для мобильного приложения

## Проблема
Файл `assets/images/banner_education_1.png` имеет размер 933KB, что слишком велико для мобильного приложения и может вызывать проблемы с загрузкой.

## Решение
Вам нужно оптимизировать изображение, уменьшив его размер. Вот несколько способов:

### Способ 1: Использование онлайн-инструментов
1. Перейдите на сайт https://tinypng.com/
2. Загрузите файл `banner_education_1.png`
3. Скачайте оптимизированную версию
4. Замените оригинальный файл на оптимизированный

### Способ 2: Использование Photoshop или других графических редакторов
1. Откройте файл `banner_education_1.png` в графическом редакторе
2. Уменьшите размер изображения до 800x600 пикселей
3. Сохраните с качеством 80% (для JPG) или используйте сжатие PNG

### Способ 3: Использование командной строки (если установлен ImageMagick)
```bash
magick convert assets/images/banner_education_1.png -resize 800x600 -quality 80 assets/images/banner_education_1_optimized.jpg
```

## Рекомендации по размеру изображений для баннеров
- Ширина: 800-1200 пикселей
- Высота: 400-600 пикселей
- Размер файла: не более 100KB

## После оптимизации
1. Замените оригинальный файл на оптимизированный
2. Перезапустите приложение
3. Баннеры должны начать отображаться корректно

## Альтернативное решение
Если у вас нет возможности оптимизировать изображение, вы можете временно использовать один из существующих баннеров:
- `assets/images/banner_promo_1.png`
- `assets/images/banner_referral_1.png`

Для этого измените файл `lib/providers/banner_provider.dart` и замените путь к изображению:
```dart
imageUrl: 'assets/images/banner_education_1.png',
```
на:
```dart
imageUrl: 'assets/images/banner_promo_1.png',
